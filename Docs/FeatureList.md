## Technical Features

### Offline-First Architecture
- All data stored locally (UserDefaults)
- No network dependency for core features
- Works in airplane mode

### Data Persistence
- MoodEntry: UserDefaults (JSON encoded)
- Manifestations: UserDefaults (JSON encoded)
- User Preferences: UserDefaults
- No Core Data required

### Privacy
- No analytics
- No tracking
- No third-party SDKs
- Local storage only
- COPPA compliant (12+ rating)
